Will I Make It

Primer proyecto en Unity. Un juego de minijuegos variados con desafíos únicos en cada nivel.

Jugar en itch.io

Descripción del Proyecto

Will I Make It consiste en una colección de minijuegos con diferentes temáticas, cada uno con sus propias mecánicas y desafíos para el jugador para poder recuperar las piezas de su nave.

PlaceHolder

Este proyecto fue mi introducción real al trabajo en equipo dentro de un entorno de desarrollo de videojuegos, aprendiendo a coordinar código con otro programador y gestionar un repositorio compartido.

Herramientas y Tecnologías

  • Unity
  • C#
  • Git / GitHub
  • Diseño de Minijuegos
